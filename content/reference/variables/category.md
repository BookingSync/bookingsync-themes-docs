# Category variable

1. TOC
{:toc}

## `promoted` value

Returns `true` if category is promoted

~~~django
{{ category.promoted }}
~~~

## `url` value

Renders URL to category

~~~django
{{ category.url }}
~~~

## `title` value

Renders category title

~~~django
{{ category.title }}
~~~

## `photos` value

Returns published category photos

~~~django
{{ category.photos }}
~~~
